## How To Enable Screen Vision (Local, Free)

You add two components:

**1. Screen Capture**
```
Programmatic screenshot (Python script, runs every few seconds or on demand)
```

**2. Vision Model**
```
Local vision AI that "reads" the screenshot
```

**Tools:**

| Component | Free Option |
|-----------|-------------|
| Screen capture | Python + mss or pyautogui |
| Local vision model | LLaVA, Moondream, Qwen-VL (run via Ollama) |
